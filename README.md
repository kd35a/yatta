# YATTA
*Yet Another Time Tracker (in AngularJS)*, or
[YATTA](http://www.youtube.com/watch?v=rW6M8D41ZWU) for short.

## Run YATTA
You can run YATTA locally:
```
$ npm install
$ node web.js
```
and then you can request <http://localhost:5000/>.
